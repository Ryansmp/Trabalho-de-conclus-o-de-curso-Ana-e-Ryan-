# Oficina360: Software para Gestão de Oficinas Mecânicas

O **Oficina360** é uma aplicação web desenvolvida como Trabalho de Conclusão de Curso (TCC) na PUC Minas, focada em otimizar a gestão operacional de oficinas mecânicas de pequeno e médio porte.

## 🚀 O Projeto
O sistema visa substituir processos manuais e desintegrados por uma solução centralizada, utilizando a metodologia **BPMN (Business Process Model and Notation)** para modelar e otimizar os fluxos de trabalho, reduzindo falhas operacionais e aumentando a transparência no atendimento.

## ✨ Principais Funcionalidades
- **Registro de Clientes**: Cadastro completo e histórico de interações.
- **Controle de Estoque**: Gestão de peças e insumos.
- **Ordens de Serviço (OS)**: Acompanhamento detalhado de serviços realizados.
- **Checklist de Entrada**: Documentação fotográfica e técnica do estado do veículo (ex: polimento de faróis) para evitar divergências.
- **Acessibilidade Móvel**: Interface responsiva para uso em smartphones e tablets diretamente no pátio da oficina.

## 🛠️ Metodologia e Tecnologias
- **Modelagem**: BPMN para análise e redesenho de processos.
- **Arquitetura**: Aplicação Web Responsiva.
- **Foco**: Redução de falhas no processo de recebimento e melhoria na comunicação com o cliente.
